# product-marketing

Product management and marketing strategy — from discovery through launch. Covers positioning, go-to-market, prioritization, metrics, copywriting, and every major PM/PMM artifact with production templates.

## Install

```bash
npx skills add LeahyCC/claude-skills@product-marketing
```

## What It Does

**When working on product strategy, marketing pages, or planning features**, this skill provides:

- **Canonical frameworks** — Dunford positioning, Torres discovery, Cagan strategy, Reforge growth
- **Production templates** — PRD, battle card, press release, OKR doc, roadmap, and more
- **Decision matrices** — which framework to use, when, and why
- **FAIL/PASS examples** — see the anti-pattern and the fix side by side
- **Marketing copy frameworks** — PAS, AIDA, BAB for landing pages, emails, changelogs

## What's Different

| Other PM/marketing skills | product-marketing |
|---|---|
| Prompts or checklists without sourcing | **Every framework cites its author and publication** |
| PM only or marketing only | **Combined PM + PMM** — positioning informs product, discovery informs marketing |
| Generic advice ("validate your idea") | **Specific templates and FAIL/PASS patterns** you can use immediately |
| No prioritization guidance | **RICE, ICE, WSJF scoring with worked examples** |
| No metrics framework | **North Star + AARRR + unit economics** with benchmarks |
